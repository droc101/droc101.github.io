Chrome Javascript Bookmarks (And some examples)

There is a way to execute JavaScript from a bookmark in Chrome. This works in Chrome 78+. It has not been tested in other browsers

How to do it

Create a new bookmark and name it anything you want.

For the URL, enter the following code:

javascript: [your code] void 0

Obviously, replace [your code] with some javascript you wrote.

What's the catch?

One thing, as far as I know:

You cannot execute it on builtin chrome pages (anything that starts with "chrome://") A.K.A. the New Tab page or settings

UPDATE: I think there is a flag to allow this

Snatch A Favicon

This is a little tool I made using google's Favicon Snatcher. Put the code as a URL in a Chrome bookmark.

javascript:var q = prompt('What domain would you like to snatch a favicon from?'); window.open('https://www.google.com/s2/favicons?domain='+q); void 0
